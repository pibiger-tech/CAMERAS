**U20-GC2053-1080PW - 产品说明**

---

## 产品概述

本款专业级 USB 摄像头采用 **2MP (1920×1080) CMOS (1/2.9") 传感器**，光学格式 **1/2.9"**，像素尺寸 **2.8µm × 2.8µm**，具备出色的感光能力与高信噪比，在复杂光照下依然成像纯净。

内置 **支持 WDR (依 GC2053 官方 datasheet)** 技术，在高对比场景下智能平衡曝光，同时保留高光与暗部细节。

基于标准 **USB 视频类(UVC)** 协议，真正即插即用，兼容 Windows, Linux, macOS, Android, Raspberry Pi, embedded systems，无需安装专有驱动。

**镜头：** YTOT YT10077-HD (Part No. 01.10077.001.02) | 接口 Mount: M12×P0.5 | 焦距 Focal length: 3.18mm ±5% | 光圈 Aperture: F1.61 ±10% (固定光圈 Fixed iris) | 调焦 Focus: 手动 Manual | 视场角 FOV @实际 1/2.9" 像面 (对角 Φ6.17mm): D≈120.5° (依镜头规格书视点表; 镜头本体最大可覆盖 1/2.7"=134°) | 像面 Image circle: Φ6.9mm (MAX) | 镜头总长 TTL: 22.12mm ±0.2 (in air) | 光学后焦 BFL: 5.37mm ±0.2 | 镜片结构 Structure: 1G5P (第一片为塑胶镜片 first element plastic) | 光学畸变 Distortion: -51% (@1/2.8") | 相对照度 Rel. illumination: 44% (@1/2.8") | 最近物距 M.O.D: 1.8m | 镜头外径 Barrel Φ14mm | RoHS

## 三、Python SDK（多平台支持）

本仓库提供一套通用 Python 工具包，支持在 **Windows、Linux、macOS** 三个平台上对本摄像头进行控制与采集。所有摄像头能力（格式、分辨率、帧率、图像控制参数）均在运行时自动发现，无需针对特定型号硬编码。

**目录：** [`python-code/`](./python-code/)

### 功能特性

| 功能 | 说明 |
| :--- | :--- |
| **摄像头自动发现** | 自动检测所有 UVC 摄像头，自动排除 CSI/MIPI 平台节点 |
| **格式枚举** | 列出所有支持的格式（MJPG / YUYV）、分辨率及帧率 |
| **最优模式自动选择** | 自动选取最高分辨率 MJPG 模式及最高帧率 |
| **曝光控制** | 手动设置曝光时间（Linux/macOS 单位 100 µs；Windows 为 log₂ 秒） |
| **增益控制** | 读写模拟增益 |
| **自动曝光** | 启用/禁用自动曝光，自适应摄像头实际菜单选项 |
| **图像控制** | 亮度、对比度、饱和度、锐度、白平衡、背光补偿 |
| **实时预览** | Linux 使用 OpenCV GUI；Windows 使用 DirectShow；macOS 使用 uvc-util |
| **无头验证** | 测量实际帧率并保存快照，无需显示器 |
| **多摄像头支持** | 可按设备路径（`/dev/videoN`）或名称字符串指定摄像头 |

### 各平台快速上手

**Linux：**
```bash
sudo apt install v4l-utils python3-opencv
python3 python-code/uvc_camera.py --list
python3 python-code/uvc_camera.py --info
python3 python-code/uvc_camera.py --verify --frames 90 --snapshot test
python3 python-code/linux/uvc_linux_control.py --status
python3 python-code/linux/uvc_linux_control.py --exposure-ms 5 --gain 16
python3 python-code/preview_linux.py
```

**Windows：**
```
python python-code\windows\uvc_windows_control.py --list
python python-code\windows\uvc_windows_control.py --status
python python-code\windows\uvc_windows_control.py --exposure-ms 10 --gain 32
python python-code\preview_windows.py
```

**macOS：**
```bash
brew install uvc-util python-tk
pip install opencv-python
python3 python-code/macos/uvc_exposure_macos.py --status
python3 python-code/macos/uvc_exposure_macos.py --exposure-ms 10
python3 python-code/preview_macos.py
```

### 曝光单位参考

| 操作系统 | 后端 | 曝光单位 | 手动切换方式 |
| :--- | :--- | :--- | :--- |
| Linux | `v4l2-ctl` | `exposure_time_absolute`，100 µs 为单位 | `auto_exposure=1` |
| macOS | `uvc-util` | `exposure-time-abs`，100 µs 为单位 | `auto-exposure-mode=1` |
| Windows | OpenCV DirectShow | `CAP_PROP_EXPOSURE` = log₂(秒) | `CAP_PROP_AUTO_EXPOSURE=0.25` |
